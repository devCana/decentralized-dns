Resolver

The trusted resolver this extension talks to. The omnibox opens <resolver>/web/<name>; the popup verifies the resolver's ed25519 signature on every answer. Discover resolver endpoints on-chain with ddns resolvers.

Pinned resolver key (recommended)

The resolver's ed25519 public key (0x…, from its /admin page or ddns resolvers). When set, the popup only shows "verified" if the answer was signed by this key — so a malicious or MITM'd resolver cannot fake a verified result. Leave blank to verify the signature without pinning an identity.
